# K-Means Clustering on Iris Dataset

## 📌 Project Overview
This project applies the **K-Means Clustering Algorithm** on the Iris dataset using two features:

- Sepal Length  
- Sepal Width  

The dataset is divided into **3 clusters** using unsupervised learning techniques.

---

## 📊 Dataset Information

The Iris dataset contains:
- 150 flower samples
- 3 species (Setosa, Versicolor, Virginica)
- 4 numerical features

For this project, we used:
- Sepal Length
- Sepal Width

---

## 🎯 Objective

- Apply K-Means clustering
- Divide the dataset into 3 clusters
- Visualize cluster separation
- Analyze grouping pattern

---

## 🧠 Algorithm Used – K-Means Clustering

K-Means works in the following steps:

1. Initialize K centroids randomly
2. Assign each data point to the nearest centroid
3. Recalculate centroids
4. Repeat until centroids stop changing (convergence)
